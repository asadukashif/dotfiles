## My Dotfiles for terminal and zsh plugins

### Required packages
- zsh
- zoxide
- fzf
- bat
- exa
- thefuck
- zinit
- git
- stow


### Brew command for all the installation is
```bash
brew install git zsh zoxide fzf bat exa thefuck zinit stow
```
